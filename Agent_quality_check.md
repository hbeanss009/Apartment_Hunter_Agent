How do I measure the success of this agent? 

Success is defined by the agent being able to clearly give the user a list of apartments and recommendations 
for listings based on the criteria user has given. More specifically the following criteria should be met - 

1. Accuracy	- Agent must only return apartments that are currently available and match 100% of the user's "must-have" filters (e.g., price, pet policy).

2. Ranking Quality - Top 3 results should objectively align with the user's weighted preferences 
(e.g., proximity to work over square footage).

3. Groundedness	- Every detail (price, address, amenities) must be directly traceable to the source scraped data with no "hallucinated" perks.
